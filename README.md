## 📖 概述

这不只是一个博客，而是一次关于**二元统一**的视觉实验。

我们既有理性思考，也有感性表达。这个项目用三重空间来呈现人性的完整：滚动页面时，你会从「白昼」进入「暗夜」，最终抵达「本源」。

**设计理念：**
| 空间 | 配色 | 体验 |
|------|------|------|
| 表世界 | 宣纸白 + 墨汁黑 | 秩序、逻辑 |
| 里世界 | 鸦青 + 月白 | 直觉、梦境 |
| 地界 | 老绢 + 金缮 | 灵性、归一 |

**开发工具：**
- **Antigravity** (Google DeepMind) — 架构设计、代码实现
- **Kiro** (Amazon) — 调试迭代、功能完善

## 🚀 安装

### 环境要求
- Node.js ≥ 18.18.0
- Notion 账号

### 步骤

```bash
# 1. 克隆仓库
git clone https://github.com/NatureBlueee/nature-blog.git
cd nature-blog

# 2. 安装依赖
npm install

# 3. 配置环境变量
cp .env.example .env.local
```

编辑 `.env.local`：
```env
NOTION_TOKEN=你的Notion集成Token
NOTION_DATABASE_ID=你的数据库ID
```

```bash
# 4. 启动开发服务器
npm run dev
```

打开 http://localhost:3000 查看效果。
